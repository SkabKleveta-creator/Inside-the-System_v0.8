# Inside the System v0.8

**Guided Logistics / Process Control Board RPG**

This repository contains the v0.8 pivot from the prior maintenance RPG proof slice into a modern guided control-board prototype.

## BLUF

Inside the System v0.8 turns logistics and business-process execution into a guided control-board RPG where the player preserves mission intent through evidence, controls, timing, ownership, recovery, and auditability.

## Frozen proof slice

The v0.7.x maintenance RPG proof slice remains conceptually frozen. It proved that maintenance-process concepts such as PMCS, MTTR, LRUs, O-Level / I-Level / D-Level authority, scan discipline, bypass decisions, escalation, and AAR can become playable mechanics.

v0.8 does **not** continue patching Eli movement or the 8x8 maintenance board.

## v0.8 core loop

**Brief → Map → Discover → Diagnose → Intervene → Validate → Recover → AAR**

## First scenario

**Priority Part Request — Readiness at Risk**

A priority requirement appears ready to move. The dashboard says supply is available, but evidence is incomplete, ownership is unclear, and the internal control cannot be trusted until the source record is checked.

## Player objective

Fulfill or stabilize the requirement without losing auditability, readiness confidence, or process control.

## Prototype features

- Entity roster
- Process / evidence / control map
- Selection-aware action cards
- Valid and blocked action explanations
- Phase-gated process flow
- Professional state metrics
- Evidence log
- Decision log
- Audit trail
- AAR panel
- System Facilitator export

## Design position

This is not a retro arcade game, a generic dashboard, or shallow points-and-badges gamification.

It is a playable, testable, auditable, and teachable decision environment.
